Обзор API

Документация описывает все доступные endpoints для управления пользователями, записями рабочего времени и графиками работы.

Управление пользователями

CRUD операции для пользователей системы, фильтрация по различным параметрам, назначение ролей и отделов.

Журнал рабочего времени

Добавление, редактирование и получение записей о рабочем времени с фильтрацией по пользователям и датам.

Графики работы

Управление графиками работы сотрудников, настройка рабочих часов и времени перерывов.

Аутентификация и безопасность

JWT-токены для доступа к API, ролевая модель доступа с проверкой авторизации.
